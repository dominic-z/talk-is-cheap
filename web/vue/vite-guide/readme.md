# 说明
vite官方文档的使用说明
https://vitejs.cn/vite3-cn/guide/

```shell
npm create vite@latest  --registry=https://registry.npmmirror.com

```

随后输入工程名称，工程类型（vue还是react），ts还是js，我选择的是vue starter

│
◇  Project name:
│  vite-project
│
◇  Select a framework:
│  Vue
│
◇  Select a variant:
│  Official Vue Starter ↗

在`Select features to include in your project:`环节，我选择了router+pinia，然后就可以生成一个亮丽的vue工程了。
